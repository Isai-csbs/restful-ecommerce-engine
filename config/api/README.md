# RESTful E-Commerce Engine & Inventory Portal

A lightweight, full-stack enterprise resource application built with native Object-Oriented PHP, a relational MySQL backend database framework, and dynamic asynchronous JavaScript (AJAX) operations on the frontend layout.

## Key Technical Features
* **RESTful System Architecture:** Decoupled back-end endpoints managing standard HTTP actions.
* **Relational Database Design:** Normalization patterns utilizing optimized indexing mechanisms for high query performance.
* **Security Optimizations:** Built-in safeguards preventing SQL injections via PDO execution boundaries and tracking secure sessions.
* **Asynchronous Front-End:** Native JavaScript state tracking making decoupled network requests without forcing document reloading actions.
